# JavaScript関連

## 使用頻度の高いプラグイン

- anime.js
  - DOM要素のアニメーションライブラリ。
- echo.js
  - lazyload（画像遅延読み込み）ライブラリ。
- lottie
  - Webアニメーションのライブラリ。
- swiper.js
  - https://github.com/nolimits4web/swiper
  - スライド要素のライブラリ。
- three.js
  - 3Dコンテンツを制作するためのライブラリ。
